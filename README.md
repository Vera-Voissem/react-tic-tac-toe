# React Tic-Tac-Toe

A React implementation of the official Tic-Tac-Toe tutorial project.

This project focuses on understanding React state management and component architecture rather than UI complexity.

---

## 🚀 Features

- Move history (time travel)
- Jump to any previous move
- Derived state logic (no redundant state)
- Immutable state updates
- Lifted state architecture
- Controlled components

---

## 🧠 What This Project Demonstrates

- Managing state with `useState`
- Lifting state up to a common parent component
- Using array immutability (`slice`, spread operator)
- Rendering dynamic lists with `map()` and `key`
- Computing derived state instead of storing redundant values
- Basic game logic implementation

---

## 🛠 Built With

- React
- JavaScript (ES6+)
- Functional components
- Hooks

---

## 📂 Project Structure

- `Game` – Manages history and current move
- `Board` – Renders squares and handles move logic
- `Square` – Individual interactive cell component

---

## 📌 Purpose

This project was built to strengthen understanding of React fundamentals, especially state flow and architecture patterns.
